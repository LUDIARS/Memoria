🎓 はじめての Memoria

1 / 6

🔔 通知の許可

Memoria は、 要約 / 日記 / 調べもの の処理が終わったときにブラウザ通知でお知らせします。 ローカルで使う場合も「処理が終わりました」 が画面のすみで届くので便利です。

下のボタンを押すと、 ブラウザが許可を聞いてきます。 そのまま許可してください。

💡 iPhone / iPad の場合: Memoria を「ホーム画面に追加」 してアプリのように開いた状態でないと、 通知許可のダイアログ自体が出せません (iOS 16.4 以降の仕様)。

📍 位置情報について

Memoria は機能ごとに位置情報を取得します:

  • 作業場所 / 食事登録 / GPS マップ ではブラウザの位置情報を使います。 これらの機能を初めて使うときに、 ブラウザが許可ダイアログを出します。
  • 継続的な軌跡記録 は iPhone の OwnTracks 連携 (ステップ 5 で案内)。
  • PC の WiFi スキャン から大まかな位置を推定する経路もあります (環境変数 MEMORIA_GOOGLE_GEOLOCATION_API_KEY を設定したとき・ Windows のみ・ 10 分間隔)。

これらの機能を使わなければ、 位置情報は一切収集・送信されません。

🚦 AI に自動でやらせる範囲を決める

Memoria は既定で、 保存と同時に AI が「要約」「分類」「カロリー推定」「日記生成」 を自動で走らせます。 claude / OpenAI などの AI をまだ用意していない場合や、 「素材だけ溜めて AI には触らせたくない」 場合は、 ここで個別に OFF にできます。

OFF にしても、 各画面の「再要約」「再分析」「日記生成」 などの手動ボタンは引き続き使えます。 ここでの設定は 設定 → AI / モデル の「🚦 AI 自動処理」 と同じ場所に保存されます。

🧩 Chrome 拡張をインストール

「拡張」 とは Chrome に新しい機能を追加する仕組みです。 ブラウザで見ているページを Memoria に保存するために、 一度だけインストールが必要です。

  1. 拡張機能ページ を開く (chrome://extensions を新しいタブで開きます)
  2. 右上の デベロッパーモード をオン
  3. 左上に表示される 「パッケージ化されていない拡張機能を読み込む」 を押し、 このリポジトリの …/Memoria/extension フォルダを選択
  4. 拡張のオプションで サーバ URL が http://localhost:5180 になっていることを確認
    💡 Tailscale や Cloudflare Tunnel 経由で Memoria を開いている場合でも、 拡張は同じ PC 内の Memoria に直接 (loopback) 接続するため、 この URL は localhost のままで OK です。
  5. 適当なページを開いて、 右上の Memoria ボタンを押すと保存できます (下の表示が「✓ 拡張を検出しました」 に変わります)
⏳ 拡張からの通信を待っています

💡 スマホで使う場合: スマホには Chrome 拡張がないので、 Memoria を「ホーム画面に追加」 して PWA として開き、 OS の共有メニューから保存します。

🌐 共有サーバ (Memoria Hub) — オプション

1 人で使う場合は、 このステップはスキップしてかまいません。

複数の PC や他のユーザーと 辞書 / 調べもの / ブックマーク を共有したい場合だけ、 共有サーバ (Hub) の URL をここに登録します。

🔒 日記 / GPS / 訪問履歴などの個人ログは、 共有サーバには送信されません。 ローカルにのみ保存されます。

後から 設定 → 📦 データ / Hub でも追加できます。

📍 OwnTracks (iPhone GPS 連携) — オプション

iPhone のアプリ「OwnTracks」 を経由して GPS 軌跡を Memoria に取り込めます。 細かい設定が必要なので、 不要な場合はそのまま次へ進んでください。

下の「合言葉を生成」 を押すと、 接続用の長い文字列が発行されます。 これを OwnTracks のパスワード欄に設定して使います。


          
詳しい手順は 設定 → 📚 セットアップ → OwnTracks を参照してください。


          
🗺 Google Maps API key — オプション

          
GPS の軌跡や食事の位置を地図で表示する際に必要な鍵です。 Google Cloud Console で発行します。

          
パソコンに gcloud コマンドがインストールされていれば、 下のボタンで自動取得できます。 ない場合は手動で入力してください。

                                                                        📖 OwnTracks GPS 取込みの設定
                                                                        
            
          
                                                                        
          
                                                                        
            
                                                                        Memoria は内蔵 MQTT broker (aedes、 port 1883) を持っているので、 別途 Mosquitto や Legatus を入れずに iPhone OwnTracks → Memoria 直結で GPS 軌跡を受けられます。
                                                                        

            
                                                                        1️⃣ Memoria 側で ingest key を生成
                                                                        
            
                                                                        軌跡ページ上部の 🔑 key ボタンを押し、 「🎲 新しい key を生成」 → 表示された 32 桁を OwnTracks のパスワードに使います (1 度だけ表示されます)。
                                                                        

            
                                                                        2️⃣ 接続経路を VPN で閉じる (推奨)
                                                                        
            
                                                                        MQTT broker をインターネットに直接晒すと位置情報が常時露出します。 Tailscale / WireGuard 等のオーバレイ VPN で「自端末同士」 に閉じてください。
                                                                        
            
                                                                          
              
                                                                        • home PC と iPhone を同じ Tailscale tailnet に入れる
                                                                          • 
              
                                                                        • broker は default で 0.0.0.0:1883 に listen — tailnet IP から見えれば OK
                                                                          • 
              
                                                                        • 厳格にしたい場合は MEMORIA_MQTT_BROKER_HOST=<tailnet IP> で bind を絞る
                                                                          • 
            
                                                                        

            
                                                                        3️⃣ iPhone OwnTracks アプリ設定
                                                                        
            
                                                                          
              
                                                                        • Mode = MQTT
                                                                          • 
              
                                                                        • Host = home PC の Tailscale IP (例 100.64.0.5)
                                                                          • 
              
                                                                        • Port = 1883
                                                                          • 
              
                                                                        • WebSocket = OFF
                                                                          • 
              
                                                                        • User = memoria など好きな名前 (MEMORIA_MQTT_USERNAME 未設定なら認証スキップ)
                                                                          • 
              
                                                                        • Password = 上で生成した ingest key
                                                                          • 
              
                                                                        • TopicID / Device = 任意 (Memoria は owntracks/+/+ を subscribe)
                                                                          • 
              
                                                                        • Reporting = Move 推奨 (バッテリ寄りなら Significant)
                                                                          • 
            
                                                                        

            
                                                                        4️⃣ HTTP モード (MQTT が使えない環境用)
                                                                        
            
                                                                        VPN が無い / FW が厳しい場合は HTTP モードでも受けられます。
                                                                        
            
                                                                          
              
                                                                        • Mode = HTTP
                                                                          • 
              
                                                                        • URL = https://<Memoria 公開ホスト>/api/locations/ingest
                                                                          • 
              
                                                                        • Password = 上で生成した ingest key
                                                                          • 
            
                                                                        
            
                                                                        HTTP モードでも ingest key (Authorization Bearer) で認証するので、 key が漏れない限り他者は POST できません。
                                                                        

            
                                                                        5️⃣ 接続中の WiFi で workplace を自動チェックイン (Electron 限定)
                                                                        
            
                                                                        Memoria デスクトップ版なら、 接続中の WiFi SSID から workplace を判定して自動チェックインできます (位置情報 permission を経由しない)。
                                                                        
            
                                                                          
              
                                                                        • 設定 → 作業場所 → 編集 → 「WiFi SSID」 欄に SSID を入れる
                                                                          • 
              
                                                                        • 「📶 現在の WiFi を追加」 ボタンで自動入力
                                                                          • 
              
                                                                        • その後 Memoria を起動すれば SSID 一致で自動チェックイン
                                                                          • 
            
                                                                        

            
                                                                        6️⃣ オプトアウト
                                                                        
            
                                                                          
              
                                                                        1. iPhone OwnTracks の publish を OFF
                                                                          2. 
              
                                                                        2. Memoria 軌跡タブ → 🔑 key → 「クリア (匿名)」 で ingest key を削除
                                                                          3. 
              
                                                                        3. broker 自体を止めたい場合は MEMORIA_MQTT_BROKER=off 環境変数
                                                                          4.

        iOS は 「ホーム画面に追加」 して PWA として開いた状態でないと
        通知許可ダイアログが出ません (16.4+ の仕様)。
      
                                                                              
      
                                                                              
        
        
      
                                                                              
      
                                                                              
      
                                                                                
    
                                                                                

    
    
                                                                                
      
                                                                                🧍 プロファイル (適正カロリー計算用)
                                                                                
      
                                                                                日記のカロリーバランス分析で使用。 未設定だと分析されません。
                                                                                
      
                                                                                
        
        
        
        
        
      
                                                                                

      
                                                                                🚶 移動速度の閾値 (作業場所判定)
                                                                                
      
                                                                                これより速い瞬間速度 (= 直前 GPS 点との距離 / 時間差) の点は「移動中」 とみなして場所タグ付けの対象から外します。 通り過ぎただけの場所を誤検出しないため。 既定 5 km/h (= 徒歩速度の上限近辺)。 0 を入れるとフィルタ無効。
                                                                                
      

      
                                                                                📝 日記の常設メモ
                                                                                
      
                                                                                日記生成時に毎回プロンプトに添える「常設メモ」。プロジェクトの背景・自分の役割・生成ルール (例: Memoria は LUDIARS の個人ツール。曖昧な作業は推測で断定して書く) などを書いておくと、毎日の日記の作業内容 / ハイライトに反映されます。空欄なら何も追加しません。
                                                                                
      
    
                                                                                

    
    
                                                                                
      
                                                                                📤 エクスポート / 📥 インポート
                                                                                
      
                                                                                ブックマークを JSON で書き出し / 取り込み (URL 重複はスキップ)。エクスポートはチェックを入れたカードのみ。
                                                                                
      
                                                                                
        
        
      
                                                                                

      
                                                                                🌐 マルチサーバ (Memoria Hub)
                                                                                
      
                                                                                辞書 / ディグ / ブックマークを共有できるハブを複数登録できます。各エントリは独立した Cernere ログインを持ち、左の topbar スイッチで個別 ON/OFF (複数同時 ON 可)。
