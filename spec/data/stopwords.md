# stopwords — ユーザー定義のストップワード

## `user_stopwords`
ワードクラウド / 傾向ページの集計から除外したい単語。

| 列 | 型 | NotNull | Default | 役割 |
|---|---|---|---|---|
| `word` | TEXT | ✓ | — | PK (lowercased) |
| `added_at` | TEXT | ✓ | UTC | |

## 使い所
- ワードクラウド生成時、 抽出後のリストから除外
- 傾向ページの上位語表示から除外
- ドメインカタログ自動分類のキーワードからは除外しない (汎用ストップワードとは別)
